# Puesta en marcha del Backend

Dentro del proyecto se encuentra una carpeta llamada **SimuladorResiduos**, la cual contiene todo el código fuente del backend, incluyendo clases, funciones y demás componentes necesarios para su funcionamiento.

## Cómo ejecutar el backend

1. Ingresar a la carpeta **SimuladorResiduos**.
2. Dentro de ella, abrir la carpeta **Ejecutable**.
3. Buscar el archivo **SimuladorResiduos.exe**.
4. Ejecutar el archivo **SimuladorResiduos.exe**.

Al iniciar la aplicación, se abrirá una ventana de consola donde se mostrará el puerto en el que se encuentra disponible la API del backend.

Mientras la consola permanezca abierta, el backend estará en ejecución y listo para recibir solicitudes desde el frontend.
